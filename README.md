A simple yet powerful Password Generator built to create secure, customizable passwords. This project demonstrates practical programming skills, clean code practices, and user‑friendly design.

✨ Features Implemented
- Random Password Generation
Generates strong, unpredictable passwords using a mix of characters.
- Customizable Length
Users can specify the desired password length.
- Character Set Options
- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special characters (!@#$%^&* etc.)
- Copy to Clipboard
- One‑click copy functionality for convenience.
- Responsive UI (if web‑based)
Works seamlessly across devices with a clean interface.
- Error Handling
Prevents invalid inputs (e.g., zero length or no character set selected).
